Issues chevron_right ISS-8429
In Progress High Priority Created on Oct 24, 2023

API Gateway Timeout Errors on Production Cluster B

Description

We are observing intermittent 504 Gateway Timeout errors affecting approximately 15% of inbound traffic on the Production Cluster B. This issue appears to be related to the recent deployment of the enhanced authentication middleware.

The logs indicate that the latency spike occurs during the JWT validation phase when communicating with the internal Identity Provider. This is causing downstream request queues to fill up, eventually leading to the timeout errors reported by the load balancer.

[ERROR] 2023-10-24 14:22:18.452 : upstream request timeout [METRICS] Latency P99: 12450ms for /v1/auth/validate

Immediate mitigation steps have included scaling the IP service pods, but the root cause in the middleware logic still needs investigation.
